# 🤝 CloudVar への貢献ガイド

CloudVar をより良くするための貢献を歓迎します！バグ報告、機能提案、プルリクエスト（PR）など、どのような形でも大歓迎です。

## 🛠 開発環境のセットアップ

1. リポジトリをフォークしてクローンします。
2. 依存関係をインストールします。
   ```bash
   npm install
   ```
3. サーバーを起動して動作確認します。
   ```bash
   npm start
   ```
4. クライアントSDKをビルドします。
   ```bash
   npm run build
   ```

## 📝 プルリクエストの送り方

1. **ブランチの作成**: `main` ブランチから新しい機能用のブランチを作成してください。 (`feat/my-new-feature` など)
2. **コードの記述**: プロジェクトの既存のスタイルに合わせ、シンプルで読みやすいコードを心がけてください。
3. **ビルドの確認**: SDKを修正した場合は、必ず `npm run build` を実行してエラーがないか確認してください。
4. **コミットメッセージ**: 日本語または英語で、変更内容がわかるメッセージを記述してください。
5. **PRの作成**: 変更内容と、なぜその変更が必要なのかを簡潔に説明してPRを送信してください。

## 💡 貢献のアイデア
- **バグ修正**: 報告されている不具合の修正。
- **ドキュメントの改善**: 誤字脱字の修正や、よりわかりやすい説明の追加。
- **新しい属性の提案**: `cv-disabled` や `cv-style` など、便利な属性の追加。
- **サンプルアプリ**: 面白い使い方のデモ作成。

## ⚖️ 行動規範
全てのコントリビューターが気持ちよく活動できるよう、建設的で敬意を持ったコミュニケーションをお願いします。
