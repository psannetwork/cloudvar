# CloudVar ☁️ 総合ドキュメント

CloudVarは、**「コードを一行も書かずにリアルタイム同期」**を目指した、最も直感的な変数同期システムです。

## 🌟 究極の「自動化」
1. **HTML自動同期**: `cv-bind`を書くだけで表示が変わる。
2. **待機がいらない**: 接続完了を待たずに代入してOK。
3. **関数がいらない**: 普通の変数代入 `=` で通信が走る。
