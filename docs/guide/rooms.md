# ルーム機能とデータ分離

CloudVar はルーム（部屋）ごとにデータが完全に分離されています。

## ルームへの参加

クライアント作成時にオプションで指定します。

```javascript
const cv = new CloudVar('ws://localhost:5032', {
  room: 'my-awesome-room'
});
```

動的にルームを変更することも可能です。

```javascript
cv.join('another-room');
```

## 自動削除
サーバー設定により、入室者が0人になったルームは **5分後** に自動的にメモリから削除されます。

## 同期の仕組み
- 同じルームにいるユーザー同士のみがデータを共有します。
- 変数 `score` を更新すると、そのルームの全員に `update` メッセージが送信されます。
- 新しくルームに入った際、サーバー上の現在のデータ（最新状態）が自動的に取得されます。
